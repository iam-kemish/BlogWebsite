# MVC Blog Website

A simple blog website built with ASP.NET MVC where users can create posts, view categories, and leave comments. It includes admin role logic and pagination.

## 🛠 Technologies
- ASP.NET MVC
- C#
- Entity Framework
- SQL Server
- Bootstrap (optional if used)

## 🚀 Features
- Create, edit, delete blog posts
- Comment on posts
- Category management
- Pagination
- Admin role logic and authorization

## 📥 Installation
1. Clone this repo  
   `git clone https://github.com/iam-kemish/BlogWebsite.git`  
2. Open the solution in Visual Studio  
3. Restore NuGet packages  
4. Update your database connection string in `appsettings.json`  
5. Run migrations (`update-database`)  
6. Press F5 to run the project
